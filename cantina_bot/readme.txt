use no terminal:
    pip freeze > requirements.txt
pra instalar as libs
No terminal, vá até o diretório do seu projeto e execute o comando:
    python app.py
Isso iniciará o servidor local Flask na porta 5000
Em outro terminal, execute:
    python interface.py
Isso abrirá a interface gráfica onde você pode adicionar, visualizar e
editar os itens do cardápio.
